Welcome to the Abbey

Abbey is a small R package with a boilerplate for creating a web application in R using the plumber and the jinjar packages.

The inspiration for the setup derives from python where you can use fastAPI and jinja2 to accomplice the same.

The boilerplate uses Bootstrap 5.3 for styling.

The package is called Abbey, because jinja means "Temple" in japanese.
